# AI-Driven Student Placement & Skill Gap Analysis

## Objective
Analyze campus recruitment data to predict student placement outcomes and identify employability skill gaps using machine learning and Power BI.

## Tools & Technologies
- Python (Pandas, NumPy, Scikit-learn)
- Machine Learning (Logistic Regression)
- Power BI
- Data Visualization

## Project Workflow
1. Data cleaning and exploratory data analysis
2. Feature engineering and skill score creation
3. Placement prediction using Logistic Regression
4. Interactive Power BI dashboard creation

## Key Insights
- Students with work experience have significantly higher placement rates
- Placed students show higher average employability skill scores
- Skill gaps strongly influence placement outcomes
